# 🤖 Ollama クイックセットアップ（5分で完了）

## 1. Ollamaをダウンロード（1分）

https://ollama.ai/download

Windows版をダウンロードしてインストール

## 2. モデルをダウンロード（3-5分）

PowerShellを開いて実行：

```powershell
ollama pull llama3.2:3b
```

## 3. 確認

```powershell
ollama list
```

`llama3.2:3b` が表示されればOK！
